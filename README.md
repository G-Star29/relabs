### Как запустить проект:

Клонировать репозиторий

```
git clone https://github.com/G-Star29/it-solutions-cars
```
Установка зависимостей:

```
pip install -r requirements.txt
```

Запуск сервера:

```
uvicorn app.main:app --reload
```
